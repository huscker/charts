# Helm Charts

[![Artifact Hub](https://img.shields.io/endpoint?url=https://artifacthub.io/badge/repository/huscker-charts)](https://artifacthub.io/packages/search?repo=huscker-charts)

A collection of Helm charts for Kubernetes.

## Usage

```bash
helm repo add huscker https://huscker.github.io/charts/
helm repo update
```

## Charts

| Chart | Description | Version |
|-------|-------------|---------|
| [s-ui](./charts/s-ui) | Web panel built on SagerNet/Sing-Box | 1.0.2 |
| [sing-box](./charts/sing-box) | Universal proxy platform (SagerNet/sing-box) | 1.0.2 |
